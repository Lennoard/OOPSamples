> _**Dada a classe abaixo:.**_

```java
public class Calculadora {
    public int soma(int op1, int op2) {
        return op1 + op2;
    }
    public double soma(int op1, int op2) {
        return op1 + op2;
    }
}
```
Qual o problema da definição acima?.

*Métodos sobrecarregados devem retornar o mesmo tipo*.
